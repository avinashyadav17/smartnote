# SmartNotes

SmartNotes is a Django-based web application for secure note-taking with CRUD functionality. Users can log in, create, view, update, and delete their personal notes.

## Features
- User registration and login
- Create, read, update, delete notes
- Protected pages using login_required decorator
- Session management
- SQLite database integration

## Technologies
- Python 3.x
- Django 3.1.12
- SQLite
- HTML, CSS, Django templates
